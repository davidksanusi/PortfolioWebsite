Commodity Dashboard

Liason International wants to explore trading commodity futures contracts as a hedge against its current physical trading operations. To better understand the market and the opportunities available, they would like a dashboard that showcases various profit forecasting models.

Objective

My task here is to build a dynamic dashboard in Google Sheet/ Excel that collects commodity pricing information and presents potential profit opportunities at a conservative rate of 10%.

Solution

I propose to build a lightweight Google Sheets dashboard that connects to a commodity API using AppScript (Excel VBAs), imports the data into the worksheet using Macros, and displays various graphs using Pivot Tables.

Software Functionality

Getting Commodity Pricing

Problem:

  • We need a way to retrieve current pricing on commodities without having to manually search and update the data ourselves.

Solution:

  • This is where AppScript, Google’s version of VBA, comes in. We can write a Javascript function that sends an API request to a commodities page and retrieves the pricing on-demand. We are then able to create a Macro that retrieves the calls and transfers the result of this function into our Google sheets table.
Current vs. Future Price Analysis

Problem:

  • We need to be able to analyze various trading scenarios where we sell at the current market price and where we sell at 10% above the current market price.

Solution:

  • My solution to this is to create multiple columns that calculate the total market value of our commodity contracts as well as the potential market value of the commodity contracts if they were to increase by 10%. This helps us see a clever picture of our dollar gain in each scenario.
Dynamic Pivot Tables

Problem:

  • We need to be able to visually analyze our commodity options and which ones we should focus on.

Solution:

  • This is where a Pivot table comes in. Once we have all the data we need, we create a pivot table to compare and contrast the performance amongst both forecasted scenarios. This gives us an outlook of the results we’d see on a relatively conservative approach. Whichever commodity reaches the target first, we will focus on capitalizing there for more opportunities.

The Benefits / Business Analysis

Sustainable

  • Since we have an automated method of getting up-to-date information on commodity prices, we can be certain that our forecasts and models will always be correct and accurate.

Scalable

  • If we ever want to add more commodities to compare and contrast, we can just add the ticker symbol to the necessary column and all the data points will be automatically populated, making the modeling process swift and easy.

Easy to Use & Comprehend

  • Our dashboard shows the compilation of various scenarios and models on a single page which allows for quick and easy analysis. If we want to make the user experience even better, we could publish this application online on the web or create a mobile app.
